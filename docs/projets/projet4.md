# Panneaux solaires montés soi-même : installation de **3kW avec batterie**

![abris-panneau](images/abris-panneau.jpg){ style="border-radius:10px ; box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);" width="500"}


### **Projet : Choisir et Comprendre les Panneaux Solaires pour une Installation Autonome**

Ce projet est né d’une expérience familiale unique. Avec mon père, nous avons installé 8 panneaux solaires de **3kW** au total, sur le toit de notre abri de jardin, et **une batterie**. Ce défi nous a permis d’apprendre énormément sur l’énergie solaire et les installations autonomes.  

Durant plus de 6 mois, mon père a compilé des informations provenant de différentes sources : sites spécialisés, forums, documentations technique. Ce travail, a permis de réunir une mine d’informations précieuses. À travers ce document, nous partageons ces connaissances et cette expérience pour aider ceux qui souhaitent mieux comprendre les panneaux solaires et envisager leur propre installation.  

#### **Ce que ce document vous propose**
- **Un retour d’expérience concret** : Découvrez comment nous avons choisi et installé nos panneaux solaires sur un abri de jardin, et les enseignements tirés de cette aventure.  
- **Des informations variées** : Profitez d’une synthèse issue de multiples sources pour mieux comprendre les caractéristiques des panneaux, les coûts, les performances et les erreurs à éviter.  
- **Une approche pratique** : Ce projet vise à rendre accessible les informations nécessaires pour évaluer la faisabilité et la rentabilité d’une petite installation solaire autonome.
